01 — Возможности

Что я умею

Django по фронтенду
Беру готовый HTML/CSS и строю под него полноценный бэкенд на Django: модели, views, шаблонизатор, деплой.
REST API на DRF
Строю API с сериализаторами, аутентификацией, пагинацией и документацией. JWT, сессии, токены.
Базы данных
SQLite для прототипов, PostgreSQL для продакшена. Миграции, оптимизация запросов, связи между моделями.
Контейнеризация
Dockerfile, docker-compose. Раздельные контейнеры для приложения и БД, volumes, переменные окружения.
Telegram-боты
Aiogram 3 и pyTelegramBotAPI. Боты с FSM, инлайн-клавиатурами, хэндлерами и интеграцией с БД.
Тестирование
Django TestCase и pytest. Юнит-тесты, тесты моделей и views, фикстуры, покрытие кода.
Парсинг
Selenium для динамических сайтов: автоматизация браузера, сбор данных, обход пагинации, сохранение в БД.

Занимаюсь бэкенд-разработкой на Python. Основной инструмент — Django и его экосистема: DRF для API, ORM для работы с базами, встроенный тест-раннер и pytest для покрытия кода.

Умею взять готовый фронтенд и построить под него весь бэкенд — от моделей до деплоя. Пишу Telegram-ботов, парсю данные с помощью Selenium, контейнеризирую проекты через Docker.

LOFT — интернет-магазин

Полноценный интернет‑магазин мебели, разработанный на Django. Реализованы регистрация и авторизация пользователей, личный кабинет, каталог товаров с категориями, карточки товаров, корзина, оформление заказов, административная панель, интеграция Stripe, работа с PostgreSQL, загрузка изображений и адаптация бизнес‑логики под реальные сценарии интернет‑магазина. Проект позволил получить практический опыт проектирования архитектуры, работы с ORM, интеграции сторонних сервисов и организации кода по приложениям.
